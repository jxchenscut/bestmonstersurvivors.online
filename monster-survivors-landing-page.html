Game Overview

Monster Survivors is an intense survival game where players must navigate through increasingly difficult waves of monsters. Utilize unique abilities, collect power-ups, and develop strategic approaches to outlast the endless hordes.

Key Features

  • • Procedurally generated monster waves
  • • Multiple character classes
  • • Real-time combat system
  • • Progressive difficulty levels
